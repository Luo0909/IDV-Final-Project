Data Description

This chart shows the concentration trends of nitrogen oxides (as nitrogen dioxide) at 14 monitoring stations from December 1, 2015 to December 1, 2016. The unit is micrograms per cubic meter (μg/m³).

Usage Instructions:

  • Click on station names in the legend to show/hide data lines for that station
  • Use the mouse wheel to zoom in/out on the chart time range
  • Drag in the chart area to pan and view different time periods
  • Hover over data points to see specific values
  • Use the control panel above to filter time and aggregate data
